Introduction

IMAGify is a user -friendly website that takes name of objects from user,then user will be able to capture some pictures for those images and then train the machine learning model to recognize images and then use live webcam rendering in the chrome browser to recognize the object or image shown through Webcam. IMAGify uses the Convolutional Neural Networks (CNN) with MobileNet architecture for image recognition which is very light-weight and can run on browsers and even mobile devices.

More about our website

The project is Web-based that uses concepts of machine learning and deep learning directly in a web browser via TensorFlow.js which is a JavaScript library for machine learning to develop ML models and use ML attributes directly in your browser. The project is basically an image classification project in real time and deployed in a browser. The browser page will render a livestream of the Webcam using webcam.js which is a JavaScript library that allows us to capture a picture using Webcam. Then we create objects from tensorflow.js data API which could capture images through webcam as tensor. Each photo is categorized into its own category and labelled. The captured images are sent to the machine learning models for training. After the training is completed the model is ready to be tested. Now if we show an image or an object through a webcam, the model will predict its label to which it belongs.
